# Game Writer
遊戲編劇可能會跟另一個職位放在一起: 敘事設計(Narrative Designer)

負責撰寫遊戲腳本，幾十年前許多遊戲只是先寫劇本再轉變成遊戲，劇情和機制沒有那麼貼合。  
現代許多遊戲編劇是要考量遊戲機制去設計世界觀和劇情

編劇是從傳統藝術小說漫畫電影等作品一路發展到遊戲的階段，所以傳統編劇的理論也要會
